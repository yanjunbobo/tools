Q: What's the principle behind timezone conversion?

A: Based on UTC (Coordinated Universal Time). Each timezone is expressed as UTC offset, e.g., Beijing is UTC+8, Tokyo is UTC+9. Conversion first converts input time to UTC, then to target timezone.Based on UTC (Coordinated Universal Time). Each timezone is expressed as UTC offset, e.g., Beijing is UTC+8, Tokyo is UTC+9.
