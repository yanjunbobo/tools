🥰 它能帮你做什么

🥰 What It Does

天数计算：想知道从哪天到哪天一共多少天？选好日期，点一下，结果秒出。勾选"排除周末"，就能算出实实在在的工作日天数！

Day Counting: Pick a start and end date, get instant results. Check "Exclude weekends" and it will crunch the actual work days for you!

倒计时模式：设置一个目标日期，马上告诉你还剩多少天。考试、假期、生日、纪念日……统统可以倒计时！

Countdown Mode: Set a target date and see exactly how many days are left. Exams, holidays, birthdays, anniversaries — count them all down!
