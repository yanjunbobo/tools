📖 使用说明User Guide

颜色代码转换教程Color Conversion Tutorial

This tool supports real-time conversion between HEX (hexadecimal), RGB and HSL color formats. Modify any format and others update automatically.

  • HEX：Such as #3B82F6, six-digit hex: first two red, middle two green, last two blue
  • RGB：Format is rgb(R, G, B), R/G/B range 0-255
  • HSL：Format is hsl(H°, S%, L%), H range 0-360, S and L range 0-100%

复制颜色教程Copy Color Tutorial

Click the color value area below the conversion result to copy the current color code with one click. A "Copied!" notification will appear on success.

💡 Tip: Colors in the palette are also auto-copied on click for quick access to common colors

渐变生成教程Gradient Generation Tutorial

The gradient generator helps you create beautiful CSS gradient effects:

  • Set start and end colors, HEX format supported
  • Adjust angle to control gradient direction (0-360°), 135° gives diagonal top-left to bottom-right
  • Click "Generate" to preview, then click the result area to copy CSS code

The generated CSS code can be used directly in web design, e.g., background: linear-gradient(135deg, #3b82f6, #8b5cf6);

❓ 常见问题FAQ

Q: Can I omit the # in HEX color code?
A: Yes, you can omit the #. Entering FFFFFF will automatically be recognized as #FFFFFF and convert normally.
Q: What happens if RGB value exceeds 255?
A: RGB values are automatically limited to the 0-255 range. Values exceeding this are truncated.
Q: What's the difference between 0° and 360° gradient angle?
A: 0° gradients from bottom to top, 360° is also bottom to top (equivalent to 0°). Angle is calculated clockwise: 90° is left to right, 180° is top to bottom, 270° is right to left.
Q: How to get more precise HSL colors?
A: Modify the S (saturation) and L (lightness) values directly in the HSL input. Higher S means more vivid color, higher L means brighter color.
